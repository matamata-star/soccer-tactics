# Tactics Board - サッカー戦術ボード

子供の練習確認用に作ったサッカー戦術ボード。ブラウザだけで動きます（ビルド不要・外部依存なし）。

## 機能

- **選手・ボールのドラッグ配置**: ボールの近くの選手を動かすとボールも追従
- **コマ送りアニメーション**: 配置をコマとして保存し、連続再生で動きを確認（ループ・速度調整・軌跡表示）
- **図形描画**: サークル・スクエア・パス矢印（実線）・走路矢印（破線）をコート上に描画。移動・リサイズ・削除可能。図形は全コマ共通
- **コート切替**: フルコート / 半面 / 無地
- **人数調整**: 各チーム1〜11人（8人制にも対応、デフォルト8人）
- **選手ラベル**: 選手をタップして番号・名前・役割を編集
- **アンドゥ**: 直前の操作を戻す
- **保存**: ブラウザ内に名前を付けて保存 / JSONファイルの書き出し・読み込み / 自動保存
- **スマホ対応**: 縦持ちでコートが自動的に縦表示に。タッチ操作対応・PWA（ホーム画面に追加してオフライン利用可）

## 使い方

`index.html` をブラウザで開くだけ。ローカルで確認する場合:

```
python -m http.server 8734
```

→ http://localhost:8734 を開く。

## 単一ファイル版

CSS/JSをすべて埋め込んだ1ファイル版を生成できます（LINEやメールで送ってスマホで開く用途）:

```
powershell -ExecutionPolicy Bypass -File build_single.ps1
```

→ `dist/soccer-tactics.html` が生成されます。

## GitHub Pages で公開する

1. GitHub でリポジトリを作成
2. このフォルダを push
3. リポジトリの Settings → Pages → Branch を `main` / `(root)` に設定
4. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます

公開URLをスマホで開き、ブラウザのメニューから「ホーム画面に追加」するとアプリのように使えます（2回目以降はオフラインでも動作）。
